# 📱 ReadLater WebApp（PWA）

スマホでも使える！シェアボタンから追加できる「あとで読む」Webアプリ

## 🌟 主な機能

### 📱 PWA対応
- **ホーム画面に追加**: アプリのように使える
- **オフライン動作**: ネットがなくても閲覧可能
- **シェアボタン対応**: 他のアプリから直接保存

### ✨ 機能
- ワンタップで保存
- 自動タグ付け
- サイト別・タグ別表示
- 検索機能
- 既読/未読管理

## 🚀 使い方

### PCでの使い方

1. **ローカルサーバーを起動**
   ```powershell
   cd C:\Users\user\.gemini\antigravity\scratch\read-later-webapp
   python -m http.server 8000
   ```

2. **ブラウザで開く**
   ```
   http://localhost:8000
   ```

3. **PWAとしてインストール**（オプション）
   - アドレスバーの右側にあるインストールアイコンをクリック

### スマホでの使い方

1. **PCと同じネットワークに接続**

2. **PCのIPアドレスを確認**
   ```powershell
   ipconfig
   ```
   IPv4アドレスをメモ（例: 192.168.1.100）

3. **スマホのブラウザで開く**
   ```
   http://[PCのIPアドレス]:8000
   ```
   例: `http://192.168.1.100:8000`

4. **ホーム画面に追加**
   
   **iPhone (Safari):**
   - 共有ボタン → 「ホーム画面に追加」
   
   **Android (Chrome):**
   - メニュー → 「ホーム画面に追加」または「アプリをインストール」

5. **シェアボタンから保存**
   - インストール後、他のアプリで記事を開く
   - シェアボタン → 「ReadLater」を選択
   - 自動的に保存されます！

## 💾 データ保存

- **LocalStorage使用**: ブラウザにデータを保存
- **端末ごとに独立**: 各デバイスで個別管理
- **クラウド同期なし**: プライバシー重視

将来的にFirebaseやSupabaseと連携すれば、複数端末での同期も可能です。

## 🔗 Chrome拡張機能との違い

| 機能 | Chrome拡張機能 | PWA |
|------|--------------|-----|
| PC対応 | ✅ | ✅ |
| スマホ対応 | ❌ | ✅ |
| 自動保存 | ✅ (Ctrl+Space) | 手動 |
| シェア対応 | ❌ | ✅ |
| オフライン | ✅ | ✅ |
| データ | Chrome Storage | LocalStorage |

## 📝 注意事項

- **ブラウザデータ**: LocalStorageを使用しているため、ブラウザのデータを削除すると消えます
- **バックアップ**: 重要なデータは定期的にエクスポート機能を使ってください（今後実装予定）
- **HTTPS必須**: PWAの一部機能はHTTPS環境でのみ動作します
